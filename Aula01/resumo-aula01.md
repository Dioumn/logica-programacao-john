# Aula 01 — Introdução à Lógica de Programação

## O que é um algoritmo?
Um algoritmo é uma sequência ordenada de passos com o objetivo de resolver um problema.

## Entrada, processamento e saída
Entrada: dados que o sistema recebe.
Processamento: transformação desses dados.
Saída: resultado produzido.

## Exemplo do dia a dia
Fazer pão assado:
1. Ir até a cozinha.
2. Localizar e coletar os objetos: Pão, faca, manteiga, frigideira e prato.
3. Usar a faca para cortar o pão ao meio no sentido horizontal.
4. Abrir o pote de manteiga.
5. Usar a faca para coletar a manteiga e passar nas faces internas do pão.
10. Posicionar a frigideira sobre a boca do fogão.
11. Colocar as duas fatias de pão dentro da frigideira, com o lado da manteiga voltado para baixo (em contato com o metal).
12. Girar o botão do fogão para abrir o gás.
13. Acionar o acendedor para iniciar a chama na boca correspondente.
14. Aguardar 2 minutos.
15. Verificar se a face em contato com a frigideira está dourada.
16. Girar o botão do fogão para a posição "Desligado".
17. Retirar o pão da frigideira utilizando a faca.
18. Colocar o pão no prato.
19. Finalizar tarefa.

## Palavras-chave
1. Algoritmo — sequência de passos para resolver um problema  
2. Entrada — informação que entra no sistema  
3. Processamento — transformação da informação  
4. Saída — resultado gerado  
5. Lógica — forma organizada de resolver problemas
